# Release v0.3.12

Date: 2026-05-26

## Summary
This version resolves an issue where review creation context remained tied to the previous organization after switching organizations in the Web UI.

We now proactively generate a new API key scoped to the selected organization during the UI context switch and persist it atomically to ~/.lrc.toml alongside org_id and org_name.

## Install and Update
New install:
```
# Linux/macOS
curl -L https://hexmos.com/ipm-install | bash && ipm i HexmosTech/git-lrc

# Windows
iwr https://hexmos.com/ipm-install-ps | iex; ipm i HexmosTech/git-lrc
```

Update:

```
lrc self-update
```
